主要功能

围绕“本地模型生命周期 + 高效交互 + 知识增强”，提供完整的桌面端能力闭环。

模型管理
扫描、导入、配置、监控
  • 自动扫描文件夹并识别模型文件
  • 参数定制：温度、上下文、批处理等
  • 加载进度、资源占用、日志与状态可视化
聊天交互
多轮对话与会话管理
  • 自然多轮对话，保持上下文连贯
  • Markdown 渲染（代码高亮、表格等）
  • 历史记录、检索、导入导出
RAG 知识库
检索增强生成
  • 多格式文档导入与智能分块
  • 向量检索 + BM25 混合检索
  • 重排序与知识注入，提升回答可信度
语音交互
TTS / STT
  • 文字转语音：多引擎可选
  • 自动播报与参数调节（音量/语速/音色）
  • 语音转文字输入（实时）
角色系统
预设 + 自定义
  • 多种预设角色：助手、翻译、程序员等
  • 支持系统提示词与行为特征配置
  • 可为角色绑定专属知识库
工具与系统集成
桌面深度集成
  • 系统托盘、通知、快捷方式与高 DPI 适配
  • 插件化工具：文件/系统/网络等能力扩展
  • 本地数据管理与迁移

典型工作流程

从“导入模型”到“知识增强对话”，几步即可开始高质量本地推理。

1) 扫描与导入
  • 选择包含模型文件的文件夹
  • 自动识别并展示模型卡片
  • 查看加载状态与资源占用
2) 配置与启动
  • 设置温度、上下文、批处理等参数
  • 选择 CPU 或 GPU 推理模式
  • 启动模型服务并开始对话
3) 构建知识库
  • 拖拽导入文档（如 PDF/TXT/网页等）
  • 自动分块、构建索引
  • 在对话中引用知识，获得更可靠回答
4) 管理会话
  • 保存历史记录并快速检索
  • 为会话打标签、分类整理
  • 导出为多种格式（以应用内为准）

FAQ

整理了微软商店用户最常问的几个问题，方便直接粘贴到商店描述或支持页。

为什么需要 runFullTrust 权限？

用于本地文件系统访问、模型文件管理、启动与监控本地 AI 进程、桌面集成能力等。具体权限说明建议在商店“功能/权限”与隐私政策中保持一致。

没有 GPU 可以使用吗？

可以。应用支持 CPU 推理，但性能取决于模型规模与硬件配置。拥有 GPU 通常能获得更流畅的体验。

支持哪些模型与格式？

支持多种主流本地模型格式（以当前版本实际实现为准）。建议在商店描述中列出已验证的格式与示例模型，并避免夸大兼容性范围。

是否会收集或上传我的数据？

本项目主张“本地优先”。最终以 隐私政策 为准：明确是否有遥测/崩溃日志、是否联网检查更新、以及用户数据的删除方式。
